A live supply chain disruption.

You don't need a CLI, an MCP server, or any local tooling. Download global-supply-chain-risk.pax.tar.gz, extract it, open the included llm-context.md, and paste it into Claude or ChatGPT as the first message. The model now has 38 constructs, 113 findings, and 47 sources on supply-chain vulnerability sitting in its context.

Then ask it real questions. The agent cites by finding id (F041) and source id (baldwin_2020) — every claim traces back to a real paper. No hallucinated stats, no made-up effect sizes.

From civil war onset to coup forecasting: state capacity as the common thread

Three pax cover the same underlying phenomenon from different angles: Fearon & Laitin's civil war onset model focuses on structural opportunity, the coup-prediction pax maps disposition and coordination among military elites, and the state-fragility pax tracks institutional erosion over time. Each uses institutional_capacity and related measures of state power.

Aligning findings across these pax lets an agent answer questions no single pax was designed for: which fragility indicators precede both coup attempts and civil war onset — and by how many years? That lead-time signal matters for early warning systems.

bridge construct
institutional_capacity
spans: fearon-laitin-2003 · military-coup-prediction · state-fragility

Praxis loads the data and runs the analysis.

The first two stories show the paste-to-Claude path. This one shows the agentic path: when an analyst is running the Praxis MCP server, an agent can do more than recall findings — it can pull the underlying datasets registered with the pax, run a registered engine on them, and return real coefficients with diagnostics.

Below: an agent loads the taxation-fiscal-policy and income-inequality-gini pax, calls praxis_get_data to pull their shared OECD-revenue / WID panel, runs an OLS regression via praxis_run_engine, and reports the coefficient table inline. Bridge construct: redistributive_effect_taxes.
